isExistSameObject

判断两个数组是否存在相同的对象 根据对象中的某一字段进行判断

Source:
  • isExistSameObject.js, line 11
Since:
  • 0.0.1
Example
isExistSameObject([{id: 1, value: 1},{id: 2, value: 2}], [{id: 2, value: 2}, {id: 3, value: 3}], 'id')
// return true

isExistSameObject([{id: 1, value: 1}], [{id: 2, value: 2}], 'id')
// return false

isExistSameObject([{id: 1, value: 1 }], {}, 'id')
// return undefined
