isObject

判断一个变量是否是对象

Source:
  • isObject.js, line 8
Since:
  • 0.0.1
Example
isObject({})
// return true

isObject([])
// return true

isObject(() => {})
// return false

isObject(null)
// return false

isObject(new Number(1))
// return true

isObject(1)
// return false
