isNumber

判断一个值是否是数值类型

Source:
  • isNumber.js, line 1
Since:
  • 0.0.1
Example
isNumber(0)
// return true

isNumber('0');
// return false

isNumber(Number('12'))
// return true

isNumber(new Number(3))
// return false
Parameters:

num any
