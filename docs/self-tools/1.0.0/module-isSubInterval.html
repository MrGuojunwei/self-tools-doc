isSubInterval

判断一个包含两个数值类型的数组区间是否在另一个包含两个数值类型的数组区间中

Source:
  • isSubInterval.js, line 3
Since:
  • 0.0.1
Example
let a = [3,7];
let b = [4,6];
let c = [1,6];
let d = [6,2];
let e = [1,9];

isSubInterval(a, []);
// return undefined

isSubInterval(a, b);
// return true

isSubInterval(a, c);
// return false

isSubInterval(a, d);
// return false

isSubInterval(e, d);
// return true

undefined参数不符合要求 true表示子数组区间在父数组区间中 false表示子数组区间不在父数组区间中
